## ðŸ“¤ UPLOADING ASSETS

### Best Practices for File Names
âŒ **BAD:** `IMG_1234.jpg`, `Copy of design.png`, `final-FINAL-v3.jpg`
âœ… **GOOD:** `hero-meauxbility-homepage-2025.jpg`, `don-wheelchair-campaign-instagram.png`

**Naming Format:** `[content-type]-[brand]-[purpose]-[date].extension`

Examples:
- `hero-meauxbility-website-nov2025.jpg`
- `social-inneranimals-product-launch.png`
- `thumbnail-youtube-accessibility-guide.jpg`
- `logo-meauxbility-horizontal-white.svg`

### How to Upload

**Method 1: Drag & Drop**
1. Navigate to **Upload Assets** in sidebar
2. Drag files from your computer directly into the upload zone
3. Select brand destination (Meauxbility, Inner Animals, or iAutodidact)
4. Wait for optimization to complete (auto-converts to WebP)
5. Done! Asset is now in your library

**Method 2: Browse Files**
1. Click "Browse Files" button
2. Select multiple files (Cmd+Click or Ctrl+Click)
3. Choose brand
4. Upload

### What Happens During Upload?
- âœ… Auto-resized to max 1920px width (maintains aspect ratio)
- âœ… Converted to WebP format (85% quality, ~70-80% smaller file size)
- âœ… Stripped of EXIF data (privacy + smaller size)
- âœ… Uploaded to Cloudflare R2 cloud storage
- âœ… CDN URL generated for instant access
- âœ… Saved to Asset Library with searchable metadata

**Optimization Example:**
- Original: `sam-pikes-peak.jpg` (4.2 MB)
- Optimized: `sam-pikes-peak.webp` (847 KB) â†’ **80% savings!**

### File Type Support
- **Images:** JPG, PNG, GIF, WebP, SVG, HEIC
- **Videos:** MP4, MOV, AVI, WebM
- **Documents:** PDF

**Size Limits:**
- Images: 50 MB max
- Videos: 500 MB max

## ðŸ“± POSTING TO SOCIAL MEDIA

### Creating a Post

**Step 1: Open Post Composer**
- Click **Schedule Post** in sidebar or quick actions

**Step 2: Select Assets**
- Browse Asset Library or search for specific images/videos
- Drag assets into composer (supports multi-image carousels)
- For Instagram: Up to 10 images
- For TikTok/YouTube: 1 video only

**Step 3: Write Caption**
- Type in the text box
- Character limits shown for each platform:
  - Instagram: 2,200 characters
  - TikTok: 2,200 characters
  - Facebook: 63,206 characters
  - LinkedIn: 3,000 characters
  - Twitter/X: 280 characters (if connected)

**Pro Tips:**
- Use hashtags! They increase reach
- Tag locations for Instagram (if relevant)
- First line is most important (hook readers)
- Emojis work great on Instagram/TikTok, less so on LinkedIn

**Step 4: Choose Platforms**
- Toggle on/off which platforms to post to
- Green = connected and ready to post
- Gray = not connected (click to connect account)

**Step 5: Preview**
- Click platform icons to see how post will look
- Instagram preview shows carousel layout
- TikTok shows video player
- LinkedIn shows professional format

**Step 6: Schedule or Post Now**
- **Post Immediately:** Click "Post Now" (publishes in ~30 seconds)
- **Schedule:** Click calendar icon â†’ Select date/time â†’ Confirm
- Posts appear in Calendar View

### Best Times to Post (Based on Analytics)

- **LinkedIn:** 7am-9am, 12pm-2pm (Tuesday-Thursday)

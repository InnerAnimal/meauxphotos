## üìä PROJECT DETAILS

**Tech Stack:**
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- Cloudflare R2 (AWS SDK)
- Supabase (Auth + Database)
- Sharp (Image optimization)

**Key Features:**
- Drag-drop upload with real-time progress
- Automatic WebP conversion (70-80% size reduction)
- Multi-brand support (3 brands)
- Social media scheduling (6 platforms)
- Team collaboration (role-based access)
- Asset library (search, filter, organize)
- Analytics dashboard
- Calendar view for scheduled posts

**Costs:**
- Vercel: Free (Hobby plan)
- Cloudflare R2: ~$0.71/month (for 50GB)
- Supabase: Free (up to 500MB DB)
- Total: **<$5/month**
